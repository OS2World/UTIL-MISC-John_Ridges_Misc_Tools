This package contains several miscellaneous tools for PM and command line.

    BEEPER - Command line tool.
    CHECKER - Error scanning program.
    DIGCLOCK - PM Clock that shows up floating.
    INDENT - Command line C file indentation cleaner tool.
    MEM - PM program to check free RAM memory.
    RENDIC - COmmand line tool to rename directories.
